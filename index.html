Afya‑Vault

A simple, fast website to securely view and record patient medical notes. Built for quick deployment using Firebase.

Why Afya‑Vault?

  • Patient‑owned records viewable anywhere
  • Doctors can create records and sign them (tamper‑evidence)
  • Simple appointment calendar and emergency card

Quick start

  1. Create a Firebase project and enable Authentication (Email/Password) and Firestore.
  2. Copy your Firebase config into config.js.
  3. Upload the site to GitHub Pages / Vercel / Firebase Hosting.

This demo is for prototyping and academic submission. Do not use it in production without security hardening.

Get started

Use the Login page to sign in. Use the Demo Dashboard to try features without creating a Firebase project.
